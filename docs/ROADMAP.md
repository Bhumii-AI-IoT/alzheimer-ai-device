# Project Roadmap

## Artificial Intelligence Enabled Device to Predict Alzheimer's Disease

---

## Where This Project Started

This project started with an observation — that Alzheimer's Disease 
is becoming one of the most difficult conditions for families to 
navigate, and that the tools available for early detection are still 
largely out of reach for most people.

That observation led to a question: whether AI and wearable technology
could give families more time to prepare. This project is an attempt
to explore that question.

---

---

## What Is Next

The immediate next step is building the signal preprocessing pipeline 
and feature extraction module — turning the simulated signals into 
structured data that a machine learning model can learn from.

Beyond that, the project will evolve based on what the work reveals.
